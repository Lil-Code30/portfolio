A modern desktop application built with WPF and the MVVM architecture that classifies wheat grain varieties using the K-Nearest Neighbors (KNN) machine learning algorithm. Features interactive prediction, large-scale experiments with configurable parameters, performance visualizations, and SQL Server persistence within a clean .NET architecture.
